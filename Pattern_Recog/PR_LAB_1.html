AIM : To plot the histogram on an image using OpenCV¶

The OpenCV Hist funtion:

cv2.calcHist(images, channels, mask, histSize, ranges[, hist[, accumulate]])
  • images : it is the source image of type uint8 or float32. it should be given in square brackets, ie, "[img]".
  • channels : it is also given in square brackets. It is the index of channel for which we calculate histogram. For example, if input is grayscale image, its value is [0]. For color image, you can pass [0], [1] or [2] to calculate histogram of blue, green or red channel respectively.
  • mask : mask image. To find histogram of full image, it is given as "None". But if you want to find histogram of particular region of image, you have to create a mask image for that and give it as mask. (I will show an example later.)
  • histSize : this represents our BIN count. Need to be given in square brackets. For full scale, we pass [256].
  • ranges : this is our RANGE. Normally, it is [0,256].

STEP 1¶

  • #### Import the required libraries
  • #### CV2 -> OpenCV
  • #### Matplotlib.pyplot -> To show images inline

import cv2
import matplotlib.pyplot as plt
%matplotlib inline

STEP 2¶

  • #### First we load the image in grayscale mode
  • #### Then calculate the histogram using the cv2.calcHist() function
  • #### Then we plot the histogram using matplotlib
  • #### Then the image is displayed
In [2]:
img = cv2.imread("totoro.jpg",0) #Read image in Gray mode

histG = cv2.calcHist([img],[0],None,[256],[0,256]) #Calculate The Hist
plt.title("Grayscale Histogram") 
plt.plot(histG, color = 'gray') #Plot the Hist
plt.show() #Display the Graph
plt.imshow(img,'gray',interpolation='nearest',aspect='auto') #Display the image

STEP 3¶

  • #### First we load the image in RGB mode
  • #### Then calculate the histogram using the cv2.calcHist() function for each channel
  • #### Then we plot the histogram using matplotlib
  • #### Then the image is displayed
In [3]:
img = cv2.imread("totoro.jpg",1)

color = ('b','g','r')
plt.title("RGB Histogram")
for i,col in enumerate(color):
    histr = cv2.calcHist([img],[i],None,[256],[0,256])
    plt.plot(histr,color = col,label=color[i])
    plt.xlim([0,256])

plt.legend(loc='upper center')
plt.show()

plt.imshow(cv2.cvtColor(img, cv2.COLOR_BGR2RGB))
